# Auth Service

A Spring Boot-based authentication microservice for my Expense Traker App  . This service is responsible for user authentication, authorization, and security management within the microservices architecture.

## Features

- User Registration
- User Login & Authentication
- JWT Token Generation & Validation
- Role-Based Access Control (RBAC)
- Secure Password Encryption
- RESTful APIs
- Integration with other microservices

## Technology Stack

- Java 17+
- Spring Boot
- Spring Security
- JWT (JSON Web Tokens)
- Maven
- MySQL/PostgreSQL
